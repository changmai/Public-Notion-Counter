📊 Public Notion Counter

Notion으로 로그인 → DB 선택 → 속성 합계 계산. 단계별로 진행합니다.

1
프록시 & 로그인
프록시 주소 확인 후 Notion 로그인
대기
로그인 후 자동으로 이 페이지로 돌아옵니다.
2
데이터베이스 연결
DB 링크/ID 입력 → 속성 불러오기
대기
로그인 완료 후 활성화됩니다.
3
합계 계산
선택한 속성의 합계를 계산합니다.
대기
속성을 먼저 선택하세요.

Tip: DB가 비어있거나 접근 권한이 없으면 속성이 보이지 않습니다. (Notion에서 해당 DB에 통합을 Add connections로 추가)
