집계하고 싶은 Notion 데이터베이스를 연결하세요.
📋 중요: 수동 연결 필수!

OAuth 로그인 후에도 각 데이터베이스마다 Integration을 수동으로 연결해야 합니다.

  1. Notion에서 집계할 데이터베이스 페이지로 이동
  2. 페이지 우상단의 "⋯" (더보기) 버튼 클릭
  3. "연결 추가" 또는 "Add connections" 선택
  4. 생성한 Integration 이름을 검색하여 연결
  5. 아래에 데이터베이스 URL 입력 후 연결 테스트

💡 팁: Integration 이름은 Notion Developer 콘솔에서 확인할 수 있습니다.
